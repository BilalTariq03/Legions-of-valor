# Card Artwork Folders

Put card images here. The game already points every card to a path based on its ID.

Recommended format:
- PNG or JPG
- Portrait card art crop
- 1024 × 768 or 768 × 1024 is fine for prototype use
- Keep filenames exactly like the `image` field in `/src/data/cards.js`

Folders:
- `/assets/cards/elves/`
- `/assets/cards/humans/`
- `/assets/cards/orcs/`
- `/assets/cards/dwarves/`
- `/assets/cards/events/`
- `/assets/cards/equipment/`
- `/assets/cards/card-backs/`

If an image is missing, the card still works. It simply shows the CSS-made placeholder art area.
